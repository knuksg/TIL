# 기초

## 1. 개요
- 파이썬은 1990년 귀도 반 로섬이 개발한 인터프리터 언어이다.

## 2. 특징
- 배우기 쉬운 언어
  - 다른 프로그래밍 언어보다 문법이 간단하면서도 엄격하지 않다.
  - 문법 표현이 매우 간결하다.
- 인터프리터 언어
  - 소스 코드를 기계어로 변환하는 컴파일 과정 없이 바로 실행 가능하다.

- 객체 지향 프로그래밍
  - 파이썬은 모든 것이 객체로 구현되어 있다.
  - 객체(object): 숫자, 문자, 클래스 등 값을 가지고 있는 모든 것.
